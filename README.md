# FPL Revenue Collection Risk Analytics Project

## Overview
End-to-end DS project analyzing customer payment delinquency and collections strategy for a utility provider.

## Tools Used
Python pandas NumPy scikit-learn Jupyter Notebook

## Source
UC Irvine Default CC Clients data (https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients)


## To run
- pip install -r requirements.txt
- open Notebook in Jupyter 
- run the sections in order
